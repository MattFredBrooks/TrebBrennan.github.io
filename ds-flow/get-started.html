Starting a new project

If you're kicking off a new project of even looking to do some prototyping in code the easiest way to get started is with the Starter Pack.

1. Download the starter pack

The starter pack contains all of the base files you'll need. Download the Starter Pack and unzip it in an appropriate project folder 2. Install dependencies In Terminal (Mac) or CommanLine (Win) Navigate to your project folder and run. > npm install 3. Install the components you need. - Head over to our download page - Select the components you want to use in your project - Copy the command line prompts 4. Using it! From here you can browser our Components page and use the HTML.
